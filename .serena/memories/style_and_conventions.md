# Style & Conventions

- Python 3.12, standard PEP 8
- Module layout follows a clear separation: capture / streaming / manager / desktop (UI) / cli — keep concerns isolated
- Root `.py` files (`desktop_app.py`, `server.py`) are compatibility shims — do not put real logic there; add to `ccfii_display_share/` package instead
- PySide6 for UI — the desktop module owns all operator-facing UI
- FFmpeg is invoked via subprocess; capture lifecycle lives in `capture.py`/`capture/`
- Tests live in `tests/` and use pytest; naming pattern `test_<subject>.py`
- No autogenerated Shopify files in this repo (global rule n/a here)
